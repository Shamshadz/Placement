System Design — HLD / LLD

Purpose
- Store high-level and low-level design notes, sample architectures, and past interview answers.

Suggested subfolders
- HLD/          — high-level design documents and diagrams
- LLD/          — component-level diagrams and sequence diagrams
- examples/     — sample designs for common systems (URL shortener, chat, etc.)

How to use
- Add concise context, constraints, and trade-offs at the top of each design document.
